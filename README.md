Weather App--
A clean, modern Weather App built using HTML, CSS, JavaScript, and the OpenWeather API.
Shows real-time weather conditions, temperature, humidity, wind speed, and more.
Technologies Used--
HTML5, CSS3 (Custom CSS), JavaScript (ES6), Fetch API, OpenWeather API, Async/Await, JSON Parsing
Setup Instructions--
1.Get a free API key from: https://openweathermap.org/
2.Open WA.js
3.Replace: const apiKey = "YOUR_API_KEY_HERE";
4.With your actual API key-Open WA.html in a browser.
